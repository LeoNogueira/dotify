# Dotify

Dotify is a simple CLI tool to make managing dotfiles on your system easy and configurable.

## Installation

Add this line to your application's Gemfile:

    gem 'dotify'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dotify
